GRF ANN

🔍 Revolutionizing GRF Measurement

Ground Reaction Force (GRF) is a crucial metric in biomechanics, sports science, and injury prevention. It is traditionally measured using expensive force plates in laboratory settings. However, not everyone has access to these high-cost tools. The GRF Artificial Neural Network (ANN) project aims to solve this problem by using wearable sensors and machine learning models to estimate GRF in real-time, without the need for force plates.

  • Eliminates the need for force plates
  • Provides cost-effective motion analysis
  • Enables real-time GRF prediction
  • Allows outdoor usage in real-world conditions

📈 Key Benefits & Applications

The GRF ANN system benefits a wide range of users in sports, healthcare, and research. By providing accurate GRF estimates without force plates, it enables real-time performance analysis and injury prevention.

🏃‍♂️ Athletes & Runners

Monitor foot impact forces to optimize performance and reduce injury risk.

👟 Coaches & Trainers

Analyze running mechanics and improve training strategies for athletes.

🦿 Physiotherapists

Evaluate patient recovery post-injury using detailed biomechanical data.

📊 Researchers

Provide an affordable alternative to force plates for studying biomechanics.

🔄 Process

🛠 Step 1: Data Collection

The system uses IMU (Inertial Measurement Unit) sensors, such as the MPU-6050, which are placed on a shoe or lower leg. These sensors collect data on acceleration, gyroscopic movement, and orientation to track motion in 3D space.

📡 Step 2: Wireless Data Transmission

The sensor data is transmitted to a central processing unit using ESP32, a Wi-Fi-enabled microcontroller. The data is sent via Wi-Fi or Bluetooth to a cloud server or a local system for analysis.

🧠 Step 3: AI-Based GRF Prediction

A trained Artificial Neural Network (ANN) processes the motion data to estimate the Ground Reaction Forces (GRF). The ANN model is trained using data from real force plates, allowing it to make accurate predictions.

  • Vertical GRF (Z-axis): Measures landing impact
  • Anteroposterior GRF (X-axis): Measures braking and propulsion forces
  • Mediolateral GRF (Y-axis): Measures stability during movement

🌍 Step 4: Web-Based Visualization

The processed GRF values are displayed in real-time on a web application. The system provides graphs, numerical values, and performance feedback.

🔄 Step 5: Feedback & Optimization

The user receives instant feedback on running efficiency, foot strike patterns, and impact forces. This helps in making immediate adjustments to improve performance and reduce injury risks.
